Digital Cashless Payment for Public Transportation

AauJam is a digital cashless payment system for public transportation. It uses RFID cards, digital wallet, mobile app, web admin panel, backend APIs, and MySQL database to make bus fare payment faster and more transparent.

Java Spring Boot MySQL Android App RFID ESP32 VPS Hosting

  • The project was hosted on a VPS with the domain aaujam.com.
